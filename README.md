## Scripts

| Script | Description |
|--------|-------------|
| system_report.sh | Displays kernel version, system info, and GPLv2 license |
| kernel_inspector.sh | Checks kernel headers and loaded modules |
| kernel_audit.sh | Audits kernel directories and permissions |
| kernel_log_analyzer.sh | Analyzes kernel logs for errors |
| kernel_manifesto.sh | Generates open-source manifesto |

## How to Run

Run these commands in command prompt:
chmod +x scripts/*.sh
./scripts/system_report.sh
./scripts/kernel_inspector.sh
sudo ./scripts/kernel_audit.sh
sudo ./scripts/kernel_log_analyzer.sh error
./scripts/kernel_manifesto.sh
